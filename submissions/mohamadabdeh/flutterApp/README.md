## About the Project
This is a simple Flutter mobile application that I made for COM206 (Visual Programming).

The app shows how to use navigation between pages, take input from the user, and update the screen.

## Tools I Used
- Visual Studio Code → for writing the code  
- Android Studio → to run the emulator  
- Flutter SDK  

## App Pages

### 1. Welcome Page
This page shows:
- My name
- My student ID
- Course name

There is also a button to start the app.

---

### 2. Home Page
This page has:
- A welcome message
- A button to go to the menu page

---

### 3. Menu Page
This page shows 5 options:
- Full Name
- Age
- Department
- University
- City

When I click on any item, it opens the next page directly.

---

### 4. Input Page
This page changes depending on what I choose.

User can:
- Write something
- Copy the text
- See the result
- Clear the input
- Go back or go home
